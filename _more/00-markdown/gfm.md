## GFM 延伸語法範例 (表格與程式碼)

### 程式碼區塊 (Code Fences)

GFM 支援使用三個反引號 (\`\`\`) 來創建程式碼區塊，並可以指定語言進行語法高亮 (Syntax Highlighting)。

#### Python 範例

```python
def fibonacci(n):
    """計算第 n 個費波那契數。"""
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

print(fibonacci(10)) # 輸出 55
```

#### JavaScript 範例

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("World"); // 輸出 "Hello, World!"
```

### 表格 (Tables)

使用**管道符號** (`|`) 和**連字號** (`-`) 來創建表格。冒號 (`:`) 用於指定對齊方式。

| 標題 1 (靠左對齊) | 標題 2 (居中對齊) | 標題 3 (靠右對齊) |
| :---------------- | :---------------: | ----------------: |
| 資料 A1           | 資料 A2           | 資料 A3           |
| 資料 B1           | 資料 B2           | 資料 B3           |
| 資料 C1           | 資料 C2           | 資料 C3           |

### 其他 GFM 延伸語法

#### 待辦清單 (Task Lists)

使用 `- [ ]` 或 `- [x]` 創建待辦清單。

  - [x] 準備 Markdown 範例
  - [x] 包含標準語法
  - [ ] 包含 GFM 延伸語法 (表格、程式碼)
  - [ ] 寄送給使用者

#### 刪除線

使用兩個波浪符號 (\~\~) 來添加\~\~刪除線\~\~。
